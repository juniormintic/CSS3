Prefijos de navegadores

se usan en casos de compatibilidad de tecnologias de estilo un caso es css grid en donde en determinados navegadores se usan anteponiendo una palabra por lo queneral ya existen herramienteas que hacen esto por nosotros como los empaquetadores para mas detalle o cambios manuales ver el siguiente link : Autoprefixer CSS online

Ejemplo

                /* 
                https://autoprefixer.github.io/
                https://caniuse.com/
                Prefijos de los navegadores
                -webkit-user-select: none;
                -moz-user-select: none;
                -ms-user-select: none;
                user-select: none;
                */

  • Navegadores de microsoft -ms-
  • Firefox -moz-
  • Todos los demas navegadores -webkit-
